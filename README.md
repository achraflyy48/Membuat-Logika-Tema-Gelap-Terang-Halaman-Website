# Simple Theme Switcher (Light/Dark Mode) 🌓

Sebuah proyek web sederhana yang mendemonstrasikan implementasi fitur peralihan tema (*theme switching*) secara dinamis. Proyek ini merupakan bagian dari eksplorasi pengembangan web untuk mengasah kemampuan dalam manipulasi DOM (Document Object Model) dan pengelolaan gaya visual menggunakan CSS Variables.

---

## 🚀 Fitur Utama

- **Dynamic Theme Toggle**: Berpindah antara tema terang (hijau-hitam) dan tema gelap (hitam-putih) secara instan.
- **Responsive Button**: Teks pada tombol berubah secara dinamis sesuai dengan tema yang aktif (menampilkan "Dark" saat di mode terang, dan "light" saat di mode gelap).
- **Modern CSS Implementation**: Menggunakan variabel CSS (`:root`) untuk mempermudah pemeliharaan warna dan konsistensi tema.
- **Clean UI**: Tata letak yang sederhana, bersih, dan fungsional.

## 🛠️ Teknologi yang Digunakan

| Teknologi | Penggunaan |
| :--- | :--- |
| **HTML5** | Struktur konten dan elemen semantik. |
| **CSS3** | Penataan gaya, tata letak absolut, dan variabel warna. |
| **JavaScript (ES6)** | Logika peralihan tema dan *event handling*. |

## 📂 Struktur File

- `index.html`: Berisi struktur utama halaman, termasuk daftar tugas dan tombol kontrol.

- `app.js`: Logika JavaScript untuk mengganti *class* pada elemen `body`.
